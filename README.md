# Pukou Home

一个用于记录、整理和分享知识的协作仓库。

我们希望把学习和实践过程中产生的有价值内容沉淀下来，让知识不只停留在个人手中，也方便其他人查阅、学习和交流。

## 内容范围

这里可以归档包括但不限于以下内容：

- 技术文章与经验总结
- 学习笔记
- 项目文档
- 工具使用指南
- 技术分享材料
- 其他值得记录和分享的内容

## 如何参与

1. 从主分支创建自己的开发分支。
2. 在对应目录中添加或更新内容。
3. 提交修改并推送到远端仓库。
4. 创建 Pull Request，检查后合并。

建议优先使用 Markdown 编写文档，并使用清晰、易于理解的文件名和目录结构。

## 一起建设

欢迎分享实践经验，也欢迎补充和完善已有内容。

每一次认真记录，都可能为后来阅读它的人节省一些时间。
